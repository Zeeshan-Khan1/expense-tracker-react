# Expense Tracker - React Application

A simple and intuitive expense tracking web application built with React.js. Track your income and expenses with real-time balance updates.

## Features
-  **Real-time Balance Calculation**: Automatically updates your balance
-  **Transaction History**: View all income and expenses
-  **Add Transactions**: Easily add new income or expense entries
-  **Simple Interface**: Clean and user-friendly design
-  **Income/Expense Summary**: See total income and expenses at a glance

## Installation & Setup

1. **Clone the repository**
```js
git clone https://github.com/yourusername/expense-tracker-react.git
cd expense-tracker-react
```
## Install dependencies
```js
npm install
```
## Run the development server
```js
npm start
```
## Open in browser
- Navigate to http://localhost:3000
